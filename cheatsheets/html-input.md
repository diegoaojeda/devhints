     <input ...
       disabled
       required
       checked

       autofocus

       autocomplete='off'        <!-- autocomplete -->
       autocompletetype='cc-exp'
       autocapitalize='off'      <!-- for mobiles -->
       pattern='\d*'             <!-- force numeric input in iOS -->

### Input types

Text:

 * email
 * hidden
 * month
 * password
 * tel
 * text
 * search
 * week

Time:

 * date
 * datetime
 * datetime-local
 * time

Etc:

 * file
 * radio
 * checkbox

Buttons:

 * button
 * reset
 * submit
 * image

Numeric:

 * number
 * range
