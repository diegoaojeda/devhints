Reference
---------
{: .-three-column}

### Store and retrieve

    $ pass insert twitter.com/rsc
    $ pass twitter.com/rsc

### Search

    $ pass find twitter.com

### Management

    $ pass mv twitter.com twitter.com/rsc
    $ pass rm [-rf] twitter.com
    $ pass cp twitter.com/rsc twitter.com/ricosc

    $ pass edit twitter.com/rsc
